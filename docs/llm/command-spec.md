# Command Spec (LLM)

Global rules:
1. Commands print one JSON object to stdout.
2. Bridge-backed commands fail with `BRIDGE_UNAVAILABLE` if bridge is down.
3. Start bridge with `harness-blender bridge start`.

Exit codes:
- `0` success
- `1` internal error
- `2` not found
- `3` validation failed
- `4` invalid input
- `5` bridge unavailable
